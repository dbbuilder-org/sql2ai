# SQL2.AI Shared Library

Shared utilities and components for SQL2.AI platform.

## Features

- Database connection utilities (PostgreSQL, MySQL, SQL Server)
- Authentication helpers
- Observability (OpenTelemetry, structured logging)
- Common Pydantic models
- Resilience patterns (circuit breaker, retry)

## Installation

```bash
pip install sql2ai-shared
```

## Usage

```python
from sql2ai_shared import ...
```
